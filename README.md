## Работа с файлами .csv, .tsv на github

1) Установить или скачать пакет git-lfs (по [ссылке](https://git-lfs.com/) есть разные варианты). Это делаете единожды
2) Выполнить команду `git lfs install` единожны
3) Если у вас уже скачан репозиторий, то нужно сделать `git pull` ветки `main` и только после этого создавать свою ветку
4) Если уже есть ветка, то нужно отребейзить ее на свежую ветку main `git rebase origin/main`
5) Чтобы локально работать с файлами `.csv` или `.tsv` как с обычными файлами, то перед работой в своей ветке выполните команду `git lfs pull`

Далее по идее, если выдобавите новый `.csv` или обновите старый, то благодаря текущим настройкам в .gitattributes эти файлы будут отслеживаться и автоматически преобразовываться в Git LFS
